# ci3-adminlte-hmvc
Base of Web Application using Codeigniter 3, HMVC, and AdminLTE Theme

#How to use
1. Import DB on "ci3-adminlte-hmvc\DB"
2. Change Config on "ci3-adminlte-hmvc\application\config" base on your server.
